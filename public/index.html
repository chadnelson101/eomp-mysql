Function and Path Summary

Function Path Description
getAllProducts / Retrieves all products from the database
getProductByID /:id Retrieves a specific product by its ID from the database
createProduct / Creates a new product with the provided details in the request body.
deleteProductById /:id Deletes a product with the specified ID from the database.
updateProduct /:id Updates the details of a product with the specified ID using the information provided in the request body.
addUser / Creates a new user account with the provided details in the request body
getUsers / Retrieves all users from the database
getUserByID /:id Retrieves a specific user by their ID from the database
deleteUser /:id Deletes a user with the specified ID from the database
loginUser /login Authenticates a user using their email and password
editUser /:id Updates the details of a user with the specified ID using the information provided in the request body.
